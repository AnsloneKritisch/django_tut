Download in Virtual Environment
--------------------------------
Note -> DJango Should be uninstall from computer before using it in Virtual Environment

Steps :- 

First we will Create Virtual Environment Wrapper
Second we will Create Virtual Environment 
Third we will Install Django 

Codes to Be use : )  -> 

First we will Install Virtual Environment Wrapper
-> pip install virtualenvwrapper-win

Second we will Install Virtual Environment Wrapper
-> mkvirtualenv environment_name

Third we will Install Django
-> pip install django

             or
How to Active the virtual environment ?

Uninstall The Virtual Environment
---------------------------------
First go inside the Virtual environment using -> workon environment_name
-> pip uninstall django

Second go out of the environment , open new cmd
-> rmvirtualen environment_name

Third Uninstall virtual environment wrapper
-> pip uninstall virtualenvwrapper-win
